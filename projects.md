#### Predicting Human Activity Data 
[Github repo](https://github.com/amalyacox/PredictHumanActivity)
This project was the final project for Stanford STATS 202: Data Mining and Analysis. Working with [Shuxian Nian](https://www.linkedin.com/in/shuxian-nian/), we used smartphone sensor data to predict different human activities. We used a support vector machine to build our final prediction model, tuning the cost parameter with 10-fold cross validation. Ultimately we obtained a 94% success rate for predicting human activity from test sensor data. Details on the data and modeling can be found in the github repo.
